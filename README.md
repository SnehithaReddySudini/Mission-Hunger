# Mission Hunger 🍱

Mission Hunger is a MERN-stack food donation platform that helps connect food donors with people in need. The application allows users to donate food, request food, and manage donations through a secure and user-friendly interface.

## 🌐 Live Demo

https://mission-hunger.onrender.com

## 🚀 Features

* User Registration and Login
* JWT Authentication
* Secure User Authorization
* Food Donation Management
* Food Request Management
* View Available Donations
* Track My Donations
* Track My Requests
* Admin Dashboard
* Responsive User Interface

## 🛠️ Tech Stack

### Frontend

* React.js

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/SnehithaReddySudini/Mission-Hunger.git
```

Install backend dependencies:

```bash
npm install
```

Install frontend dependencies:

```bash
cd client
npm install
```

Run the application:

```bash
npm run dev
```
